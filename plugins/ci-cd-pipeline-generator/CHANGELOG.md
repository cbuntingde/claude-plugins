## [1.0.0] - 2026-01-17

### Added
- Initial release of CI/CD Pipeline Generator plugin
- GitHub Actions workflow generation
- GitLab CI pipeline generation
- Jenkins pipeline generation
- Automatic project detection and analysis
- Support for multiple programming languages (Node.js, Python, Java, Go, Rust, PHP)
- Docker build and deployment support
- Kubernetes deployment integration
- Dependency caching strategies
- Parallel job execution
- Security scanning integration (SAST, SCA, container scanning)
- Multi-environment deployment (dev, staging, production)
- Monorepo pipeline templates
- Microservices pipeline templates
- GitOps workflow support (ArgoCD, Flux)
- Pipeline Architect agent for complex scenarios
- Auto-invoked skill for automatic CI/CD generation
- Comprehensive documentation and examples

### Supported Platforms
- GitHub Actions
- GitLab CI/CD
- Jenkins
